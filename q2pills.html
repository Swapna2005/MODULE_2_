Tabbed Navigation Tabbed navigation is a way to navigate around a website. Normally, tabbed navigation uses navigation buttons (tabs) arranged together with the selected tab highlighted. This example uses elements with the same class name ("city" in our example) , and changes the style between display:none and display:block to hide and display different content
